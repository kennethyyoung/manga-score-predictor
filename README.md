# Anime Score Predictor

## Overview
This project uses Python and Scikit-learn to predict anime scores based on the MyAnimeList dataset.

## Key Findings
* Utilized Linear Regression to identify that [Factor X] is the strongest predictor of score.
* Corrected data leakage found in the `members_log` feature.

## Technologies
* Python
* Pandas
* Scikit-learn (Linear Regression, KNN)
